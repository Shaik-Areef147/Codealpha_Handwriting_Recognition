# Handwritten Character Recognition using CNN

## 📌 Overview

Handwritten Character Recognition is a Machine Learning and Deep Learning web application that recognizes handwritten digits from uploaded images using a Convolutional Neural Network (CNN). The application provides an intuitive, responsive, and visually appealing interface where users can upload handwritten images and receive instant predictions with confidence scores.

The project is built using the MNIST dataset for training and leverages TensorFlow/Keras for the deep learning model. A Flask backend handles image preprocessing and prediction, while a modern HTML, CSS, and JavaScript frontend ensures a smooth and interactive user experience.

---

## 🚀 Features

* Upload handwritten digit images for prediction.
* Deep Learning model built with Convolutional Neural Networks (CNN).
* High prediction accuracy using the MNIST dataset.
* Displays predicted digit along with confidence score.
* Fast and efficient image preprocessing.
* Attractive and responsive user interface.
* Smooth Flask backend integration.

## ⚙️ Project Workflow

1. User uploads a handwritten digit image.
2. The Flask server receives the image.
3. The image is converted to grayscale and resized to 28×28 pixels.
4. Pixel values are normalized.
5. The trained CNN model predicts the handwritten digit.
6. The prediction and confidence score are displayed instantly on the website.
